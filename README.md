# residence-itylon

## Start the API

Go in the API folder
`cd api`

Run the start command
`npm start`

Go to your browser, to `localhost:8002`

Example:

`../api-docs`: to get the API swagger

## Start the FRONTEND

Go to the FRONT folder
`cd front`

Run the start command
`npm run dev`

Go to the browser, to `http://localhost:5173/`
